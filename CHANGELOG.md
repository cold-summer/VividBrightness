# Changelog

All notable changes to VividBrightness are documented in this file. The project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Changed

- Prepared the repository for public GitHub development.
- Replaced ambiguous EDR-headroom status text with the selected brightness boost.

## [1.0.0] - 2026-07-21

### Added

- Native macOS menu bar interface with continuous and preset boost controls.
- XDR/EDR display detection and 16-bit floating-point Metal overlay rendering.
- Wake restoration, global keyboard shortcut, and clean overlay shutdown.
- Hardware integration probe for validating real EDR headroom.
