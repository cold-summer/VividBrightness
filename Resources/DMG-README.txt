VividBrightness - First Run / 首次运行说明
============================================

English
-------
1. Drag VividBrightness.app to the Applications folder in this window.
2. This build is ad-hoc signed and is not Apple-notarized. Open Terminal and run:

   xattr -dr com.apple.quarantine /Applications/VividBrightness.app
   open /Applications/VividBrightness.app

You only need to remove the quarantine attribute once after installation.

简体中文
--------
1. 将 VividBrightness.app 拖到本窗口中的 Applications 文件夹。
2. 此版本使用临时签名，未经 Apple 公证。打开“终端”并执行：

   xattr -dr com.apple.quarantine /Applications/VividBrightness.app
   open /Applications/VividBrightness.app

安装后只需移除一次隔离属性。
